Añadiendo contenido a la tabla

En este otro ejemplo partimos de una tabla de tres filas y tres columnas.

El contenido de las diversas celdas se ha de introducir entre las etiquetas <td> y </td>.

Si se trata de un texto se introduce directamente, pero si se trata de una imagen se ha de utilizar la etiqueta <img src="">, con el nombre del archivo de imagen dentro de las comillas.

Fíjate, a continuación, en el código html utilizado para insertar la tabla de tres filas y tres columnas.

Para las imágenes se puede introducir también el nombre de archivo pezsargo.jpg y pezlimon.jpg.

En el espacio del texto se puede poner cualquier cosa.
