Imágenes animadas

En este ejemplo sobre el canvas se desplaza una rana verde, que aparece y desaparece. Para ello utilizamos de forma indefinida dos ordenes que nos permiten desplazar el origen de coordenadas del canvas y girarlo.

Para desplazar el origen de coordenadas se utiliza la orden ctx.translate(0, 0); y para girar el canvas la orden ctx.rotate(-0.25);.

La orden ctx.clearRect(0, 0, canvas.width, canvas.height); nos permite borrar el canvas antes de dibujar una nueva rana. Para que todo esto ocurra de forma indefinida se utiliza la orden setInterval(dibujarcanvas, 100); que llama a la función dibujarcanvas() cada 100 milisegundos.

Haz todas las modificaciones que creas oportunas en el dibujo, para familiarizarte con el código. Modifica las dimensiones de la rana, el ángulo de giro y los cambios de coordenadas.
