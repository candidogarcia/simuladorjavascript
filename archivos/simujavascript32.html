Vamos a mover un punto de derecha a izquierda

En este ejemplo la función funcionmover(desplazamiento) se utiliza para cambiar la imagen del punto rojo de derecha a izquierda y viceversa.

Dependiendo del valor de la variable desplazamiento se consigue que el punto se desplace hacia un lado o hacia el otro.

Haz todas las modificaciones que creas oportunas, para familiarizarte con el código.

 
<html>
<head>
<title>primera página</title>
<script type="text/javascript">

function funcionmover(desplazamiento){

if(desplazamiento==1){
document.getElementById("izquierda").src="puntorojo.jpg";
document.getElementById("derecha").src="fondoazul.jpg";
}

if(desplazamiento==2){
document.getElementById("izquierda").src="fondoazul.jpg";
document.getElementById("derecha").src="puntorojo.jpg";
}

}

</script>
