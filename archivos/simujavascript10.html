Enlazando otros contenidos

Los enlaces son unos de los elementos más característicos de los contenidos web. Nos permiten ir de un documento a otro y de un contenido a una serie interminable de ellos.

Las etiquetas que nos permiten hacerlo son <a href:""> y </a>. Entre estas dos etiquetas se situa el texto o la imagen que queremos enlazar.

Al hacer clic sobre este texto o imagen se abren los nuevos contenidos. La URL en donde se encuentran estos contenidos se ha de colocar en la primera etiqueta, entre las comillas.

Esta etiqueta utiliza el atributo target para determinar si el nuevo contenido se abrirá en la misma ventana o en una nueva ventana. El atributo target ="_blank" hace que el enlace se abra en una nueva ventana, por el contrario target = "_self" abre el contenido en la misma ventana. Si no se utiliza este atributo el contenido enlazado se abre en la misma ventana.
