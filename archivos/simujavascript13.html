Hojas de estilo CSS

Las hojas de estilo CSS son conjuntos de instrucciones que se asocian a un texto y le proporcionan formato. Estas hojas de estilo se sitúan en la cabecera de la página web para su uso posterior. El código de estas hojas de estilo se introduce entre las etiquetas <style> y </style>.

En este ejemplo tenemos tres hojas de estilo texto1, texto2, texto3 texto4 y texto5.

Para asignar una hoja de estilo a un párrafo hemos de introducir este atributo dentro de la etiqueta del mismo de esta forma <p class="texto3" />.

También se usan hojas de estilos para dar formato a las tablas, las imágenes o todo el cuerpo del documento web.

En este ejemplo podemos modificar el texto utilizando, como hemos dicho, las hojas de estilo CSS, que se pueden incluir en una etiqueta <p> o <blockquote>.

 
<html>
<head>
<title>primera página</title>
<style type="text/css">
.texto1 {
font-family: Georgia, "Times New Roman", serif;
font-size:32px;
color: red;
}
.texto2 {
font-family: Helvetica, Geneva, Arial, sans-serif;
font-size:12px;
color: green;
}
.texto3{
font-family:Arial, Helvetica, sans-serif;
font-size:18px;
color:#08088A;
}
.texto4{
font-family:Arial, Helvetica, sans-serif;
font-size:18px;
color:#08088A;
text-align: right;
}
.texto5{
font-family:Times New Roman;
font-size:16px;
color:#000000;
text-align: left;
} </style>
</head>
<body>
<blockquote>
<p>

</p>
<p>

</p>
<p>
<blockquote align="right">

</blockquote>
</p>
<p>

</p>

<p>

</p>
<p class="texto2" />

</p>
</body>
</html>


La Vía láctea

La Vía Láctea es una galaxia espiral en donde se encuentra el Sistema Solar.

El nombre Vía Láctea proviene de la mitología griega y en latín significa camino de leche.

Se calcula que contiene entre 200.000 millones y 400.000 millones de estrellas.

La distancia desde el Sol hasta el centro de la galaxia es de alrededor de 27.700 a&ntilte;os luz (8.500 pc), es decir, el 55 % del radio total galáctico.

La Vía Láctea forma parte de un conjunto de unas cuarenta galaxias llamado Grupo Local, y es la segunda más grande y brillante tras la Galaxia de Andrómeda (aunque puede ser la más masiva, al mostrar un estudio reciente que nuestra galaxia es un 50 % más masiva de lo que se creía anteriormente).

La mitología griega explicaba que se trata de leche derramada del pecho de la diosa Hera (Juno para los romanos). Rubens representa la leyenda en su obra El nacimiento de la Vía Láctea.
