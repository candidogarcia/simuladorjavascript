Ejercicio final de formato de texto

Ahora vamos a realizar este ejercicio desde cero. En la ventana de texto has de ir introduciendo los diferentes párrafos con el código necesario para dar formato al texto, utilizando también hojas de estilo CSS.

Para empezar una etiqueta <blockquote> nos permitirá introducir un tabulador que afectará a todo el texto.

Ahora hemos de introducir el título "Los delfines", utilizando la hoja de estilo CSS texto1, mediante la etiqueta <p class="texto1"/>.

En el primer párrafo se ha de introducir este texto: "Los delfínidos (Delphinidae), llamados comúnmente delfines oceánicos, son una familia de cetáceos odontocetos muy heterogénea, que comprende 34 especies.", utilizando la hoja de estilo CSS texto5.

Ahora se vuelve a introducir una etiqueta <blockquote> que nos permitirá poner un nuevo tabulador.

En el segundo párrafo se ha de introducir este texto: "Miden entre 2 y 9 metros de largo, con el cuerpo fusiforme y la cabeza de gran tamaño, el hocico alargado y solo un espiráculo en la parte superior de la cabeza.", utilizando la hoja de estilo CSS texto4.

Ahora mediante la etiqueta </blockquote> eliminamos un tabulador.

En el tercer párrafo se ha de introducir este texto: "Están entre las especies más inteligentes que habitan el planeta. Se encuentran relativamente cerca de las costas y a menudo interactúan con el ser humano.", utilizando la hoja de estilo CSS texto2.
