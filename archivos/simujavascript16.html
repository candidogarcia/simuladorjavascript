Entramos en JavaScript

Este es el primer documento web en el que aparece lenguaje JavaScript.

El código de JavaScript se ha de introducir en la cabecera del documento entre las etiquetas <script type="text/javascript"> y </script>.

En este ejemplo JavaScript hace que se abra una ventana emergente, mediante el código alert, cada vez que se carga la página.

La función ventanaemergente() es quien hace que se abra esa nueva ventana. El evento onload, que indica que la página ya se ha cargado, en la etiqueta body es quien llama a la función ventanaemergente().

 
<html>
<head>
<title>primera página</title>
<script type="text/javascript">
function ventanaemergente(){
alert("¡Bienvenido! acabas de entrar en el mundo de JavaScript");
}
</script>
</head>
<body onload ="ventanaemergente()">

</body>
</html>

Este es el inicio de JavaScript
