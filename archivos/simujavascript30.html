Funciones condicionales y variables

En este ejemplo vamos a trabajar con funciones condicionales, pero utilizando una doble condición. Dentro del paréntesis de la función condicional se pueden enlazar diversas condiciones con los caracteres && si se han de cumplir las dos condiciones a la vez, o los caracteres || si basta con que se cumpla una de las dos condiciones. Se ha de tener un cuidado especial para no olvidar ningún signo para abrir o cerrar paréntesis.

En este caso disponemos de cuatro flechas que llaman a las funciones funcionvariableuno(indiceuno) y funcionvariabledos(indicedos) y nos permiten aumentar o disminuir el valor de las variables variableuno y variabledos.

Al final de las dos funciones funcionvariableuno(indiceuno) y funcionvariabledos(indicedos) se muestran los valores de las variables, mediante la orden document.getElementById("muestravariableuno").innerHTML = variableuno; y se llama a la función funcioncomparar().

La función funcioncomparar() tiene dentro de ella 6 funciones condicionales que comprueban una doble condición cada una. Estas funciones condicionales no agotan todas las posibilidades, por lo que puedes buscar otras que hagan que se muestren otros posibles mensajes.

Haz todas las modificaciones que creas oportunas, para familiarizarte con el código.

 
<html>
<head>
<title>primera página</title>
<script type="text/javascript">

var variableuno = 0;

var variabledos = 0;

function funcionvariableuno(indiceuno){

if(indiceuno==1){
variableuno = variableuno - 1;
}

if(indiceuno==2){
variableuno = variableuno + 1;
}

document.getElementById("muestravariableuno").innerHTML = variableuno;

funcioncomparar();

}

function funcionvariabledos(indicedos){

if(indicedos==1){
variabledos = variabledos - 1;
}

if(indicedos==2){
variabledos = variabledos + 1;
}

document.getElementById("muestravariabledos").innerHTML = variabledos;

funcioncomparar();

}

function funcioncomparar(){

var mensaje ="";

if((variableuno<10)&&(variableuno>variabledos)){
mensaje = "La variableuno es menor que 10 y mayor que variabledos."
}

if((variableuno>10)&&(variableuno>variabledos)){
mensaje = "La variableuno es mayor que 10 y mayor que variabledos."
}

if((variableuno==10)&&(variableuno>variabledos)){
mensaje = "La variableuno es igual a 10 y mayor que variabledos."
}

if((variableuno<10)&&(variableuno<variabledos)){
mensaje = "La variableuno es menor que 10 y menor que variabledos."
}

if((variableuno>10)&&(variableuno<variabledos)){
mensaje = "La variableuno es mayor que 10 y menor que variabledos."
}

if((variableuno==10)&&(variableuno<variabledos)){
mensaje = "La variableuno es igual a 10 y menor que variabledos."
}

document.getElementById("muestramensaje").innerHTML = mensaje;

}

</script>
