Ancho y alto: redimensionar elementos

En nuestro documento web podemos modificar las medidas de imágenes, margenes y celdas, utilizando las etiquetas <height> para fijar la altura y <width> para determinar la anchura.

Estos atributos se han de introducir dentro de las etiquetas correspondientes a imagenes o celdas.

Aunque hayamos fijado las dimensiones de una celda, las imágenes, si lo necesitan, harán que se expandan en anchura y altura. Por este motivo conviene redimensionar las imágenes para que se limiten a ocupar el espacio que les hayamos asignado. Los textos se expanden solamente en vertical.

En este ejemplo podemos modificar todo el contenido y la distribución de los elementos de la tabla y sus dimensiones.

 
<html>
<head>
<title>primera página</title>
<style type="text/css">
.texto1 {
font-family: Georgia, "Times New Roman", serif;
font-size:32px;
color: red;
}
.texto2 {
font-family: Helvetica, Geneva, Arial, sans-serif;
font-size:12px;
color: green;
}
.texto3{
font-family:Arial, Helvetica, sans-serif;
font-size:18px;
color:#08088A;
}
</style>
</head>
<body>

</body>
</html>
