JavaScript es un lenguaje de programación creado para proporcionar funciones adicionales y dinamismo a las páginas web. Tiene algunas semejanzas con los lenguajes C y Java.

JavaScript se utiliza junto con el lenguaje en el que se escriben las páginas web, el html.
