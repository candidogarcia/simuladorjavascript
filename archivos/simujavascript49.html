Eventos de teclado

En este ejemplo dentro del canvas se mueve una imagen. Para conseguirlo se utiliza el evento de teclado "onkeydown". Este evento se inicia en la etiqueta <body>, cuando se presiona una tecla.

El evento llama a la función "whichButton(event)", a la que envía el valor "event". Según sea este valor, es decir, según cual sea la tecla presionada entrará en uno de los cuatro condicionales. Con esto se consigue modificar las coordenadas "x" e "y" de la imagen que se muestra en el canvas.

La imagen se mueve con las cuatro teclas de dirección (Las cuatro flechas del teclado).

Haz todas las modificaciones que creas oportunas en la animación, para familiarizarte con el código.

 
<html>
<head>
<title>primera página</title>
</head>
<body onkeydown="whichButton(event)" onload="lienzo()">

<canvas id="micanvas" width="500" height="500" style="border:1px solid #000000;background-color:#87CEEB;">
</canvas>
</p>
<script type="text/javascript">

var x=0, y=0;//coordenadas iniciales de la imagen

function lienzo(){

var c=document.getElementById("micanvas");
var ctx=c.getContext("2d");
var img1 = new Image();
img1.src = 'colibri.jpg';
ctx.clearRect(0,0,500,500);//limpia todo el canvas
ctx.drawImage(img1,x,y,50,50);//dibuja la imagen

} //cuando presiona una tecla
function whichButton(event){

if(event.keyCode=='39'){//si la tecla presionada es direccional derecho

x=x+5;//mueve la nave 5 pixeles a la derecha
lienzo();

}

if(event.keyCode=='37'){//si la tecla presionada es direccional izquierdo

x=x-5;//mueve la nave 5 pixeles a la izquierda
lienzo();

}

if(event.keyCode=='38'){//si la tecla presionada es direccional arriba

y=y-5;//sube la nave
lienzo();

}

if(event.keyCode=='40'){// si la tecla presionada es direccional abajo

y=y+5;//baja la nave
lienzo();

}

}

</script>

</body>
</html>
