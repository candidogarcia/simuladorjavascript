Una calculadora con cuadros de texto

En este ejemplo tenemos otra calculadora con dos cuadros de texto y un teclado con las cuatro operaciones básicas.

Un cuadro de texto es un espacio en donde podemos escribir para introducir información en una variable, aunque también puede ser un espacio en donde se muestre el resultado. Cuando escribimos o borramos algo en uno de estos cuadros de texto del ejemplo, se llama a la función introducirnumero(ordennumero), que guarda el valor del cuadro de texto en la variable numeroprimero o numerosegundo.

Cuando se hace clic sobre una tecla de operación se llama a la función operar(indice) en donde, mediante cuatro funciones condicionales, se obtiene el valor de la variable resultado.

Haz todas las modificaciones que creas oportunas, para familiarizarte con el código.

 
<html>
<head>
<title>primera página</title>
<script type="text/javascript">

var numeroprimero = "";

var numerosegundo = "";

function introducirnumero(ordennumero){

if(ordennumero==1){
var numeroprimero = document.getElementById("primernumero").value;
document.getElementById("primernumero").innerHTML = numeroprimero;
}

if(ordennumero==2){
var numerosegundo = document.getElementById("segundonumero").value;
document.getElementById("segundonumero").innerHTML = numerosegundo;
}

}

function operar(indice){

var numeroprimero = document.getElementById("primernumero").value;
var numerosegundo = document.getElementById("segundonumero").value;

var numerouno = parseInt(numeroprimero);
var numerodos = parseInt(numerosegundo);

if(indice==1){
var resultado = numerouno + numerodos;
}

if(indice==2){
resultado = numerouno - numerodos;
}

if(indice==3){
resultado = numerouno * numerodos;
}

if(indice==4){
resultado = numerouno / numerodos;
}

document.getElementById("resultados").innerHTML = resultado;

}

</script>
</head>
