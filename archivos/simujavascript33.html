Mover el punto también arriba y abajo

En este ejemplo hemos de conseguir mover el punto rojo en las cuatro direcciones, en una cuadrícula de 2x2.

Para determinar la posición del punto rojo se utilizan las variables horizontal y vertical. Asignando valores a horizontal y vertical se puede situar la posición en cada uno de los caudrados de la cuadrícula.

Las cuatro flechas llaman a la función funcionmover(desplazamiento) que modifica los valores de las variables horizontal y vertical. Una vez hecho esto llama a la función borrar() y coloca la imagen fondoazul.jpg en todas las casillas. Una vez hecho esto llama a la función dibujarpunto() que situa la imagen puntorojo.jpg en donde corresponde.

Haz todas las modificaciones que creas oportunas, para familiarizarte con el código. A continuación intenta hacer una página similar sobre una cuadrícula de 3x3.

 
<html>
<head>
<title>primera página</title>
<script type="text/javascript">

var horizontal = 1;

var vertical = 1;

function funcionmover(desplazamiento){

if((desplazamiento==1)&&(vertical==2)){
vertical = 1;
}

if((desplazamiento==4)&&(vertical==1)){
vertical = 2;
}

if((desplazamiento==2)&&(horizontal==2)){
horizontal = 1;
}

if((desplazamiento==3)&&(horizontal==1)){
horizontal = 2;
}

borrar();

}

function borrar(){
document.getElementById("puntouno").src="fondoazul.jpg";
document.getElementById("puntodos").src="fondoazul.jpg";
document.getElementById("puntotres").src="fondoazul.jpg";
document.getElementById("puntocuatro").src="fondoazul.jpg";

dibujarpunto();

}

function dibujarpunto(){

if((horizontal==1)&&(vertical==1)){
document.getElementById("puntouno").src="puntorojo.jpg";
}

if((horizontal==2)&&(vertical==1)){
document.getElementById("puntodos").src="puntorojo.jpg";
}

if((horizontal==1)&&(vertical==2)){
document.getElementById("puntotres").src="puntorojo.jpg";
}

if((horizontal==2)&&(vertical==2)){
document.getElementById("puntocuatro").src="puntorojo.jpg";
}

}
</script>
</head>
