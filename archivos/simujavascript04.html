Filas y columnas

En este otro ejemplo partimos de una tabla de cuatro filas y cuatro columnas.

Hemos de recordar que para insertar tablas se utilizan las etiquetas <table> y </table>. Las etiquetas <tr> y </tr> delimitan una fila de la tabla. Por último, las etiquetas <td> y </td> sirven para definir una celda dentro de una fila.

Fíjate, a continuación, en el código html utilizado para insertar la tabla de cuatro filas y cuatro columnas.

Modificando la cantidad de bloques de código <tr><td></td><td></td><td></td><td></td></tr> podemos modificar la cantidad de filas de la tabla.

Eliminando o volviendo a poner estos bloques de código, mediante los desplegables, podemos conseguir que haya desde una hasta cuatro filas.

 
<html>
<head>
<title>primera página</title>
</head>
<body>
<table>
<tr>
<td>
</td>
<td>
</td>
<td>
</td>

</td>
</tr>



</table>
</body>
</html>
