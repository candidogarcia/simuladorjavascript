Funciones condicionales

En este ejemplo vamos a trabajar con funciones condicionales. Este tipo de estructura es fundamental en cualquier lenguaje de programación y también en JavaScript.

Las funciones condicionales permiten al programa (Al código) decidir que es lo que se ha de hacer, en función del valor de una, o de más de una, variable.

La estructura de una función condicional es esta: if(){... }. Dentro del paréntesis se ha de escribir la condición que se ha de cumplir para que se ejecute el código que colocamos entre las llaves. En este caso la condición que se ha de cumplir es que el valor de la variable cantidad sea igual a uno, dos, tres o cuatro. Esto se comprueba con un doble igual (cantidad==1).

En este ejemplo tenemos un teclado con cuatro números. Cuando hacemos clic sobre uno de ellos llamamos a la función funciongatos(cantidad) y además le enviamos el valor de la variable cantidad. Dependiendo del valor de esta variable cantidad hacemos que aparezca una imagen diferente en la imagen con identificador gato.

Haz todas las modificaciones que creas oportunas, para familiarizarte con el código.

 
<html>
<head>
<title>primera página</title>
<script type="text/javascript">

function funciongatos(cantidad){

if(cantidad==1){
document.getElementById("gato").src="ungato.jpg";
}

if(cantidad==2){
document.getElementById("gato").src="dosgatos.jpg";
}

if(cantidad==3){
document.getElementById("gato").src="tresgatos.jpg";
}

if(cantidad==4){
document.getElementById("gato").src="cuatrogatos.jpg";
}

}

</script>
</head>
<body>
<p>

</p>
</body>
</html>
