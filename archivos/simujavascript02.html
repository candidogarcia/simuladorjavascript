Como insertar una imagen

Para insertar imágenes en una página web se utiliza la etiqueta <img src="">. Dentro de las comillas se coloca el nombre del archivo de imagen, con su extensión, o la dirección URL de la imagen.

El código correspondiente a las imágenes también se ha de colocar dentro del body, en la zona en la que colocamos las cosas que queremos que el navegador muestre en pantalla.

Fíjate, a continuación, en el código html utilizado para insertar imágenes y cambia el nombre del archivo de imagen o la URL de la misma. Para poder ver los cambios se ha de hacer un clic fuera del cuadro de texto que hemos modificado.

Los archivos de imagen que puedes utilizar son: perro.jpg, gato.jpg y loro.jpg. También puedes utilizar la URL de cualquier imagen que te encuentres en internet.

 
<html>
<head>
<title>primera página</title>
</head>
<body>
<img src=" ">
</body>
</html>
