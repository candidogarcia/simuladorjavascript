Trabajando con cadenas

En este ejemplo vamos a trabajar con cadenas. Una cadena es una variable alfanumérica, que puede contener letras y números. Se puede modificar una cadena eliminando algunos de sus caracteres.

Si nos fijamos, veremos que se usan dos funciones funcionescribir(indice) y funcionborrar(). Cuando desde el teclado tecleamos sobre una letra llamamos a la función funcionescribir(indice) y además le enviamos el valor de la variable indice. Del valor de la variable indice depende la letra que se añade a la cadena cadena.

Dentro de la función funcionescribir(indice) hay cinco funciones más, de las llamadas condicionales. Las órdenes que van dentro de estas funciones se ejecutan si se cumple la condición que se encuentra dentro del paréntesis. En la función if (indice==1) se ha de cumplir que el valor de la variable indice sea igual a uno, para que se ejecute la linea de código letra = "a";.

La función funcionborrar() calcula primero la cantidad de caracteres de la cadena, mediante el código var longitud = cadena.length;, a continuación se elimina el último carácter, mediante el código cadena = cadena.substring(0, longitud-1);.

Haz todas las modificaciones que creas oportunas, para familiarizarte con el código.

<script type="text/javascript">

var cadena="";

var letra = "";

function funcionescribir(indice){

if(indice==1){
letra = "a";
}

if(indice==2){
letra = "b";
}

if(indice==3){
letra = "c";
}

if(indice==4){
letra = "d";
}

if(indice==5){
letra = "e";
}

cadena = cadena + letra;
document.getElementById("marcador").innerHTML=cadena;
}

function funcionborrar(){
var longitud = cadena.length;
cadena = cadena.substring(0, longitud-1);
document.getElementById("marcador").innerHTML=cadena;
}

</script>
