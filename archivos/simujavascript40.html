Girando una imagen sobre si misma

En este ejemplo dentro del canvas se mueven tres engranajes de forma coordinada. Para empezar podemos modificar las dimensiones de este canvas.

A continuación, podemos escoger otras imagenes para hacerlas girar. También podemos modificar las coordenadas de las imágenes y la velocidad a la que giran con la orden rotarimagen(primeraimagen,100,100,contador/1);. Si a contador le quitamos o le añadimos el signo menos conseguimos cambiar el sentido de giro. Podemos modificar individualmente la velocidad de giro de una imagen cambiando el uno que divide a contador por otro valor. Tambien podemos cambiar la velocidad de giro modificando el valor de setInterval(loop, 10);.

Haz todas las modificaciones que creas oportunas en la animación, para familiarizarte con el código.

function rotarimagen(imagen, x, y, angulo) {

context.save();
context.translate(x, y);
context.rotate(angulo * radian);
context.drawImage(imagen, -(imagen.width/2), -(imagen.height/2));
context.restore();

}
