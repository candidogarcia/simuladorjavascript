Hacer lo mismo de forma más sencilla

Al cargar una página web también se ejecuta todo el código JavaScript que se encuentre fuera de una función cualquiera. Para ejecutar una de estas funciones ha de haber algo que la llame.

En este ejemplo JavaScript hace que se abra una ventana emergente, mediante el código alert, cada vez que se carga la página, pero ya no está dentro de una función y se ha eliminado el evento de la etiqueta body.

 
<html>
<head>
<title>primera página</title>
<script type="text/javascript">
alert("¡Bienvenido! acabas de entrar en el mundo de JavaScript");
</script>
</head>
<body>

</body>
</html>

Este es el inicio de JavaScript
