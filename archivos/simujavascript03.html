Elaboración de tablas

Para insertar tablas en un documento html se utilizan tres tipos de etiquetas. Las etiquetas <table> y </table> se utilizan para identificar toda la tabla. Las etiquetas <tr> y </tr> delimitan una fila de la tabla. Por último, las etiquetas <td> y </td> sirven para definir una celda dentro de una fila.

Determinando la cantidad de filas, y de celdas por cada fila, podemos hacer que la tabla tenga más o menos casillas.

Fíjate, a continuación, en el código html utilizado para insertar una tabla con una fila y cuatro celdas dentro de esa fila. Modificando la cantidad de parejas de etiquetas <td> y </td> podemos modificar la cantidad de celdas en la fila.

Eliminando o volviendo a poner pares de etiquetas <td> </td>, mediante los desplegables, podemos conseguir que haya desde una hasta cuatro celdas.

Si en nuestra página web ponemos solamente el código que aquí aparece no veríamos nada. En este código falta el contenido de texto de la celda, la anchura de la tabla y la indicación de que se han de ver las lineas de la tabla. Si lo hemos simplificado es para ayudar a comprender como es la estructura básica de una tabla.
