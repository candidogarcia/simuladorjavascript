Elijamos una función

En este ejemplo utilizamos el botón para modificar un texto situado en un párrafo concreto.

Podemos variar el texto del botón, el evento, que podemos sustituir por onclick, ondblclick, onmousedown, onmouseup, onmouseover, onmousemove y onmouseout, y la función a la que se llama.

Al actuar sobre el botón llamamos a una función. En el cuadro de texto podemos elegir entre las funciones funcionuno(), funciondos() y funciontres().

Las tres funciones modifican el texto que se sitúa en el párrafo identicado como "parrafo", mediante la etiqueta <p id="parrafo">.

 
<html>
<head>
<title>primera página</title>
<script type="text/javascript">

function funcionuno(){
document.getElementById("parrafo").innerHTML="Este es el primer texto";
}

function funciondos(){
document.getElementById("parrafo").innerHTML="Este es el segundo texto";
}

function funciontres(){
document.getElementById("parrafo").innerHTML="Este es el tercer texto";
}

</script>
</head>
<body>
<p>

</p>
<p>

</p>
<p id="parrafo">
Texto que cambia
</p>
</body>
</html>
