Uso del canvas para mover el punto rojo

En este ejemplo volvemos a mover el punto rojo por una cuadrícula de 2 X 2, pero lo haremos utilizando otros recursos de JavaScript. Ahora vamos a utilizar un canvas, un elemento gráfico de html que se puede modificar mediante código JavaScript.

Un canvas es un espacio de dibujo en el que se pueden trazar lineas, polígonos, círculos y también posicionar imágenes. En este caso se trata de mover por el canvas la imagen del punto rojo. Detrás de ella se mantiene inmóvil la imagen de fondo.

Haz todas las modificaciones que creas oportunas, para familiarizarte con el código.
